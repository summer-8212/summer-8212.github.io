## 语音交互功能

### 使用方法

#### 1. 启动语音输入
1. 点击页面右下角的「AI成长陪伴」按钮，打开聊天窗口
2. 点击输入框旁的麦克风图标 🎤
3. 首次使用时会弹出浏览器权限请求，点击「允许」
4. 麦克风图标变为红色，开始录音
5. 说话完成后，再次点击麦克风图标或等待自动停止
6. 识别结果会自动填入输入框

#### 2. 权限设置
**Chrome浏览器：**
- 点击地址栏左侧的 🔒 图标
- 找到「麦克风」选项
- 选择「允许」

**Safari浏览器：**
- 打开「设置」→「网站」
- 找到当前网站
- 将麦克风权限设为「允许」

**微信内置浏览器：**
- 点击右上角「...」→「设置」
- 开启「使用麦克风」权限

### 常见问题

#### Q1: 点击麦克风没有反应
**排查步骤：**
1. 检查是否使用HTTPS或localhost访问
2. 检查浏览器权限设置
3. 刷新页面重试
4. 尝试使用Chrome浏览器

#### Q2: 语音识别不准确
**解决方案：**
1. 确保环境安静，减少背景噪音
2. 说话时靠近麦克风
3. 使用标准普通话
4. 说话速度适中，不要太快

#### Q3: 录音自动停止
**原因：**
- 检测到长时间静音会自动停止
- 单次录音最长60秒

**解决方案：**
- 保持连续说话
- 如需长时间录音，可分多次进行

## AI虚拟引导员

### 功能说明

AI虚拟引导员是一个软萌治愈系的卡通漂浮形象，具有以下特性：

#### 1. 悬浮定位
- 默认显示在页面右下角
- 支持拖拽到左侧或右侧
- 拖拽后会自动吸附到边缘

#### 2. 交互功能
| 操作 | 效果 |
|-----|------|
| 点击头像 | 打开AI聊天窗口 |
| 拖拽头像 | 改变位置（左/右） |
| 点击「-」按钮 | 最小化/恢复头像 |
| 长按拖拽 | 调整位置，释放后自动吸附 |

#### 3. 动画效果
- **呼吸漂浮**：头像持续上下轻微浮动
- **眨眼**：眼睛会定期眨动
- **说话动画**：AI回复时嘴巴会动
- **开心动画**：点击时显示开心表情

#### 4. 状态保存
- 位置偏好（左/右）自动保存
- 最小化状态自动保存
- 刷新页面后恢复上次状态

### 使用方法

#### 移动位置
1. 鼠标/手指按住头像
2. 拖动到想要的位置
3. 释放后自动吸附到左边缘或右边缘

#### 最小化
1. 鼠标悬停在头像上
2. 点击右上角的「-」按钮
3. 头像变为小尺寸
4. 再次点击可恢复

## 多性格人设切换

### 4套人设介绍

| 人设 | 名称 | 适用阶段 | 特点 | 音色 |
|-----|------|---------|------|------|
| 👩‍👧 | 温柔亲子陪伴型 | 一阶·亲子破局 | 共情力强、温柔耐心 | 温暖女声，语速偏慢 |
| 🧘‍♀️ | 活力正念引导型 | 二阶·身体筑基 | 积极有活力、正向鼓励 | 元气女声，节奏感强 |
| 📚 | 沉静成长教练型 | 三阶·自我整合 | 沉稳理性、共情包容 | 沉稳女声，语速偏缓 |

### 切换方法

#### 在AI聊天窗口中切换
1. 点击AI虚拟引导员头像，打开聊天窗口
2. 在聊天窗口顶部查看「当前人设」
3. 点击下方的4个选项切换不同人设
4. 切换后AI头像外观会相应变化

#### 切换效果
- **外观变化**：头像颜色、配饰图标会改变
- **语气变化**：AI回复的语气风格会改变
- **音色变化**：语音播报的音色会改变
- **话术变化**：问候语、鼓励语等内容会改变

### 人设偏好保存
- 选择的人设会自动保存
- 下次访问时自动恢复上次选择
- 不同设备需要重新设置

## 异常排查方案

### 语音交互异常

#### 问题1: 麦克风无法启动
**错误表现：**
- 点击麦克风无反应
- 控制台显示权限错误
- 提示"无法访问麦克风"

**排查步骤：**

1. **检查安全上下文**
   ```
   确认访问地址：
   ✓ https://anxinclub.com
   ✓ https://localhost
   ✓ http://localhost:8080
   ✗ http://192.168.x.x (非安全上下文)
   ```

2. **检查浏览器权限**
   - Chrome: 地址栏 🔒 → 网站设置 → 麦克风 → 允许
   - Safari: 设置 → 网站 → 麦克风 → 允许
   - Firefox: 地址栏 🎤 → 允许

3. **检查系统权限**
   - Windows: 设置 → 隐私 → 麦克风 → 允许应用访问
   - macOS: 系统偏好设置 → 安全性与隐私 → 麦克风
   - iOS: 设置 → 隐私 → 麦克风

4. **测试麦克风**
   - 打开系统录音机测试麦克风是否正常
   - 检查是否有其他应用占用麦克风

**解决方案：**
```javascript
// 在浏览器控制台运行测试
navigator.mediaDevices.getUserMedia({ audio: true })
  .then(stream => {
    console.log('麦克风权限获取成功');
    stream.getTracks().forEach(track => track.stop());
  })
  .catch(err => {
    console.error('麦克风权限获取失败:', err.name, err.message);
  });
```

#### 问题2: 语音识别无结果
**错误表现：**
- 录音正常但无文字输出
- 长时间显示"识别中..."
- 识别结果为空

**排查步骤：**

1. **检查网络连接**
   - 语音识别需要网络连接
   - 检查网络是否稳定

2. **检查浏览器支持**
   ```javascript
   // 在控制台检查支持情况
   console.log('SpeechRecognition支持:', 'SpeechRecognition' in window || 'webkitSpeechRecognition' in window);
   console.log('SpeechSynthesis支持:', 'speechSynthesis' in window);
   ```

3. **检查语言设置**
   - 确保浏览器语言包含中文
   - 尝试清除浏览器缓存

**解决方案：**
- 刷新页面重试
- 更换Chrome浏览器
- 检查网络连接

#### 问题3: 语音播报失败
**错误表现：**
- AI回复后没有声音
- 声音播放中断
- 音色不符合预期

**排查步骤：**

1. **检查音量设置**
   - 系统音量是否开启
   - 浏览器标签页是否静音
   - 网站是否有播放权限

2. **检查语音合成支持**
   ```javascript
   // 检查可用语音
   speechSynthesis.getVoices().forEach(voice => {
     console.log(voice.name, voice.lang);
   });
   ```

**解决方案：**
- 点击页面任意位置激活音频（浏览器自动播放策略）
- 检查系统音量设置
- 尝试使用耳机

### AI虚拟引导员异常

#### 问题1: 头像不显示
**排查步骤：**
1. 检查控制台是否有JS错误
2. 确认 `ai-avatar.js` 文件已加载
3. 检查是否有CSS样式冲突

**解决方案：**
```javascript
// 手动初始化
if (typeof AIAvatar !== 'undefined') {
  const avatar = new AIAvatar({
    onClick: () => openAIChat()
  });
}
```

#### 问题2: 拖拽后位置不保存
**原因：**
- 浏览器禁用了localStorage
- 无痕/隐私模式

**解决方案：**
- 检查浏览器设置
- 退出隐私模式
- 清除浏览器数据后重试

#### 问题3: 动画卡顿
**排查步骤：**
1. 检查设备性能
2. 关闭其他占用资源的标签页
3. 检查CSS动画性能

**解决方案：**
- 在移动端减少动画效果
- 使用CSS硬件加速
- 关闭其他应用

### 人设切换异常

#### 问题1: 切换后无变化
**排查步骤：**
1. 检查控制台是否有错误
2. 确认 `ai-personality.js` 已加载
3. 检查localStorage是否可用

**解决方案：**
```javascript
// 手动切换人设
if (aiPersonality) {
  aiPersonality.setPersonality('gentle'); // gentle | energetic | calm | relaxed
}
```

#### 问题2: 偏好设置不保存
**原因：**
- 浏览器禁用了localStorage
- 存储空间已满

**解决方案：**
- 清除浏览器缓存
- 检查localStorage使用情况
- 使用隐私模式测试

## 浏览器兼容性

### 完全支持
| 浏览器 | 版本 | 语音交互 | AI头像 | 人设切换 |
|-------|------|---------|--------|---------|
| Chrome | 90+ | ✅ | ✅ | ✅ |
| Edge | 90+ | ✅ | ✅ | ✅ |
| Safari | 14.1+ | ✅ | ✅ | ✅ |
| Firefox | 88+ | ⚠️ | ✅ | ✅ |

### 部分支持
| 浏览器 | 限制说明 |
|-------|---------|
| Firefox | 语音合成支持有限，可能使用默认音色 |
| Safari iOS | 需要用户交互才能播放语音 |
| Chrome Android | 语音合成质量取决于系统TTS |
| 微信内置 | 部分功能受限，建议使用系统浏览器 |

### 不支持
| 浏览器 | 原因 |
|-------|------|
| IE 11 | 不支持现代Web API |
| 旧版Android WebView | 缺少Speech API支持 |

### 推荐配置
**最佳体验：**
- Chrome 100+ (Windows/macOS/Android)
- Safari 15+ (iOS/macOS)
- 稳定的网络连接
- 麦克风权限已授权
